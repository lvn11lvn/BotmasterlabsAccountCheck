Для запуска:

Установите nodejs

Инициализируйте npm:
npm init -y

Установите express:
npm install express

Запустите server.js:
nodejs server.js

Установите Python (использовался 3.13)

Установите requests и loguru:
pip install requirements.txt

Измените login и password в check.py и запустите (из директории папки проекта):
py check.py
